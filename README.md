# 🧮 Simple Calculator Application

A simple and user-friendly calculator application built using Python.  
This program performs basic arithmetic operations based on user input.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧠 Input validation for numbers  
- ⚠️ Handles division by zero errors  

---

## 🛠️ Tech Stack

- Python  

---

## 📦 Requirements

No external libraries are required.  
This project runs using standard Python libraries.

---

## ▶️ How to Run

1. Clone the repository:
```bash
git clone <your-repo-link>
cd calculator-app
```

2. Run the program:
```bash
python main.py
```
